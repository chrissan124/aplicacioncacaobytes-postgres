## Cacao Admin API

## Requirements

- **Node.js**
  You can download the lasted Node version [here](https://nodejs.org/es/)

## Setup

- Install dependencies:

```shell
npm i
```

- Setup .env file:

  You should follow `.env.example` to see required variables

## Scripts

- Run the project locally:

```shell
    npm run dev
```

- Build only:

```shell
    npm run build
```

- Run build with nodemon:

```shell
    npm run watch:dev
```
